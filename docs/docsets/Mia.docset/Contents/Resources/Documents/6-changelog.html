Changelog

1.0.0

Released: 01-March-2020

Release notes:

  • First official release after successful pilot phase
  • For any bug reports or question, please contact: mobile-acceptance@nets.eu
  • Next releases will include wrappers for Flutter, React Native and Xamarin. We will soon also show in our sample application, how to handle recurring payments. Stay tuned!

1.1.0

Released: 08-April-2020

  • Added example and documentation for recurring/subscription payments.

1.2.0

Released: 01-June-2020

  • Added Xamarin.Android and Xamarin.iOS wrappers for SDK integration.
  • Added a shared React Native wrapper for both iOS and Android SDK integration.

1.3.0

Released: 20-Apr-2021

  • Added support for cancel URL in SDK API, allowing users to close the Nets Easy checkout (using the “Go back” button) and come back to the app or target URL. Just like returnURL, cancelURL it is a string to specify in the register payment API.
  • Updated Xamarin and ReactNative wrapper for SDK integration.

1.4.0

Released: 09-Aug-2021

  • Added app switch suppor for MobilePay.

1.5.0

Released: 27-Aug-2021

  • Added app switch support for Vipps. Refer the ‘Digital Wallets’ section in the payment flow.

1.6.0

Released: 05-Nov-2021

  • Renew the brand to Nets Easy.
  • Documentation updates.

1.6.1

Released: 07-Mar-2024

  • Added SDK privacy manifest and signatures followed by Apple third-party SDK requirements starting in spring 2024 to submit a new or updated App that adds SDKs as part of the update in App Store Connect.
